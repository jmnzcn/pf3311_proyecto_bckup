ExperimentPrototypeB03230 - build para evaluación (PF-3311)
================================================================

Requisitos
----------
- Windows 10/11, pantalla 1920×1080 (ventana maximizada).
- Internet estable (condiciones B y C: chat Gemini + voz Azure).

Ejecución
---------
1. Descomprimir toda la carpeta (no mover solo el .exe).
2. Verificar que exista _config/LocalSecrets.json junto al .exe (incluido en el zip de entrega).
3. Ejecutar ExperimentPrototypeB03230.exe (un solo clic).
4. Consentimiento → elegir INICIAR (A, B o C) → 6 preguntas por bloque.

Condiciones
-----------
- A: sin agente
- B: agente de texto (Gemini)
- C: agente virtual + voz (Gemini + Azure TTS)

Credenciales
------------
Las API keys (Gemini, Azure Speech, Drive upload) están en _config/LocalSecrets.json,
NO en el código fuente del repo. Este zip de entrega las incluye para que B y C funcionen.

Generado: junio 2026. Compartir solo con evaluadores/participantes autorizados.

Datos
-----
Las respuestas se guardan en CSV data/ junto al ejecutable.
